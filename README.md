# 🔐 PHP Login & Registration System

## 📌 Project Overview
This is a basic **Login and Registration System** built with **PHP** and **MySQL**. 
It allows users to register, log in securely, and interact with a simple authentication workflow. 

This project was created to demonstrate backend web development skills and to serve as a portfolio project.

---

## 🚀 Features
- User registration with form validation
- User login authentication
- Session management
- Secure password handling
- Database integration with MySQL

---

## 🛠️ Technologies Used
- PHP
- MySQL
- HTML & CSS
- XAMPP (for local server)
